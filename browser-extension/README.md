# DeadlineIQ Companion Chrome Extension 🧠

This browser extension integrates your active browsing workspace directly with your DeadlineIQ client app.

## Features
- **⚠️ Deadline Breach Warning**: Alerts you of upcoming critical risks (e.g. *DBMS project due in 18h*) right in your toolbar popup.
- **⏱️ Instant Focus Lock**: Clicking **Start Focus Mode** opens your local dashboard page and launches a Pomodoro Tunnel Vision session.
- **🖱️ Select & Save (Right-Click Context Menu)**: Select any assignment prompt or syllabus text on a page, right-click, and choose **Save to DeadlineIQ** to add it instantly to your inbox.
- **📂 Tab Capture**: Auto-populates active tab URLs and titles into task forms automatically.

## How to Load in Google Chrome
To test this extension during development or present it to hackathon judges:

1. Open **Google Chrome** and navigate to `chrome://extensions/`.
2. Toggle the **Developer mode** switch in the top-right corner.
3. Click the **Load unpacked** button in the top-left corner.
4. Select the `browser-extension` folder inside this project root directory (`/Users/amoteesh/Desktop/deadlineiq/browser-extension`).
5. Pin the **DeadlineIQ Companion** icon to your Chrome toolbar!
